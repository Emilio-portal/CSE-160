# CSE 160 Assignment 1 – WebGL Paint

A WebGL drawing app with a custom fox face.

## Features
- Draw points, triangles, and circles with adjustable color/size
- Drag to paint with smooth stroke interpolation
- Eyedropper tool to sample colors off the canvas
- "Draw Fox" button renders my fox face (28 triangles)

## Fox Initials
- **E** – the two pointed ears
- **P** – :P mouth with tongue sticking out

## Files
- `asg1.js` – main program, WebGL setup, event handlers
- `Point.js / Triangle.js / Circle.js` – shape classes
- `Fox.js` – fox face geometry
- `fox_sketch.jpeg` – planning sketch on graph paper
